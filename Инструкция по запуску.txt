# Инструкция по запуску
Версия Java - 22.0.2.
Версия Maven - 3.9.8.

Сборка проекта
Для сборки проекта выполните следующие шаги:

Перейдите в каталог вашего проекта (где находится файл pom.xml).

Выполните команду Maven для сборки проекта:

mvn clean package
Эта команда создаст исполняемый JAR-файл в директории target.

Запуск приложения
После сборки вы можете запустить приложение с помощью команды:

java -jar target/filedatafilter-1.0-SNAPSHOT.jar -s -a -p sample- in1.txt in2.txt

Опции:

-s: Включает краткую статистику.
-a: Включает режим добавления (append mode).
-f: Включает полную статистику.
-p: Префикс для выходных файлов.
-o: Указывает, что выходные файлы должны быть записаны в каталог по пути.
in1.txt in2.txt: Входные файлы для обработки.
